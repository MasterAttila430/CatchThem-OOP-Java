CatchThem-OOP-Java
A dynamic 2D arcade game built in Java. This project focuses on Object-Oriented Programming (OOP) principles and basic game mechanics.

Features
Object-Oriented Design: Clean and modular code using classes for the player, items, and game engine.

Collision Detection: Custom logic for real-time interaction between the player and falling objects.

Resource Management: External assets like images and icons are managed within the res directory.

Highscore System: Includes a system to track and save player performance.

Technologies Used
Language: Java

IDE: IntelliJ IDEA

Tools: Git

How to Run
Clone the repository to your local machine.

Open the project in IntelliJ IDEA.

Ensure your JDK is correctly configured.

Navigate to the src directory and run the Main class.
